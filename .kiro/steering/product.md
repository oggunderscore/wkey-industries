# Product Overview

WKEY Industries is a modern web platform showcasing three core services:

- **Photos**: Photo & Video Management service (photos.wkey-industries.net)
- **Cloud**: File Storage & Collaboration platform (cloud.wkey-industries.net)
- **Map**: Interactive Mapping Service (map.wkey-industries.net)

The main website serves as a landing page and service portal with Firebase authentication integration. The design follows the vanish-ui design system with a dark theme, purple accent colors (#a855f7), and smooth animations.

## Key Features

- Static site generation for fast loading
- Firebase authentication for user management
- Responsive design with modern UI/UX
- Netlify deployment optimized
- Shared design system (vanish-ui) for consistent branding
